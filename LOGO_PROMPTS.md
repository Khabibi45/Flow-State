# Logo Prompts pour FlowState

Ces prompts sont prets a copier-coller directement dans ChatGPT (DALL-E) ou tout autre generateur d'images par IA. Chaque variation explore un style different pour le logo FlowState — dashboard de productivite gamifie au design ultra-moderne, theme sombre, accent violet (#7c3aed).

---

## Prompt 1 — Minimaliste Geometrique

```
Design a minimal geometric logo for "FlowState", a modern productivity app. The logo is a diamond/rhombus shape inspired by the ◆ symbol. Clean sharp lines with a smooth purple gradient from #7c3aed (deep violet) to #a78bfa (soft lavender). The diamond should feel precise and balanced, with subtle inner facets suggesting depth and dimension. Place it on a pure dark background (#0f0f23). Apple-like simplicity and restraint — no unnecessary elements. The shape should work perfectly at small sizes (favicon) and large sizes (splash screen). Style: vector, flat design with gradient depth, geometric, ultra-clean. Technical: 1024x1024 pixels, transparent background, SVG-friendly clean edges. Negative: no text, no photorealistic textures, no drop shadows, no busy details, no 3D rendering.
```

## Prompt 2 — Flow / Mouvement

```
Create an abstract logo representing "flow state" and deep productivity focus. The shape is a smooth, continuous flowing curve — like a ribbon of energy or a sine wave transforming into an infinity-like loop. The form suggests effortless movement, momentum, and unbroken concentration. Use a rich gradient flowing from #7c3aed (violet) through #6366f1 (indigo) to #a78bfa (soft lavender), giving the curve a luminous quality. Dark background (#0a0a1a). The overall impression should be calm yet dynamic, like focused energy in motion. Minimal composition — just the single abstract flowing shape, perfectly centered. Style: vector, minimal, smooth bezier curves, modern tech branding, Apple-inspired restraint. Technical: 1024x1024, transparent background, SVG-friendly. Negative: no text, no literal objects, no photorealistic rendering, no complex gradients, no clutter.
```

## Conseils d'utilisation

### Pour de meilleurs resultats avec ChatGPT / DALL-E

- **Generer plusieurs variations** : lancer chaque prompt 3 a 4 fois et selectionner les meilleurs elements de chaque resultat.
- **Iterer** : prendre un resultat prometteur et demander des modifications specifiques ("make the diamond sharper", "increase the glow intensity").
- **Combiner** : si le Prompt 1 donne une belle forme et le Prompt 7 un bel eclairage, fusionner les descriptions dans un nouveau prompt.
- **Specifier le ratio** : pour DALL-E 3, preciser "square format 1:1" dans le prompt si necessaire.
- **Upscale ensuite** : utiliser un outil comme Vectorizer.ai ou Adobe Illustrator Image Trace pour convertir le resultat en vrai SVG.

### Palette de couleurs de reference

| Nom | Hex | Usage |
|-----|-----|-------|
| Violet principal | `#7c3aed` | Couleur d'accent primaire, CTA, logo |
| Lavande | `#a78bfa` | Accents secondaires, hover states, degradés |
| Indigo | `#6366f1` | Accents tertiaires, ombres colorees |
| Fond sombre | `#0f0f23` | Background principal du dashboard |
| Fond profond | `#0a0a1a` | Background alternatif, sections profondes |
| Blanc | `#ffffff` | Texte principal, symboles sur fond sombre |
| Gris sombre | `#1a1a2e` | Texte sur fond clair, elements secondaires |

### Formats de fichiers recommandes

- **SVG** : format principal pour le web (scalable, leger, editable en CSS)
- **PNG 512x512** : favicon et icones systeme (avec transparence)
- **PNG 1024x1024** : app stores, Open Graph, partage social
- **PNG 192x192 + 32x32** : PWA manifest icons et favicon.ico
- **ICO** : favicon classique (inclure 16x16, 32x32, 48x48)

### Workflow recommande

1. Generer avec les prompts ci-dessus (DALL-E ou Midjourney)
2. Selectionner le meilleur candidat
3. Vectoriser avec Vectorizer.ai ou Illustrator
4. Nettoyer le SVG (supprimer les paths inutiles, optimiser avec SVGO)
5. Exporter dans tous les formats necessaires
6. Integrer dans le projet : remplacer le `◆` actuel par le logo SVG
