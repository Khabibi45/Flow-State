# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-03-15

### Added
- **Gamification System**: XP points, 30 levels, rank progression
- **22 Badges**: Unlockable achievements for productivity milestones
- **Daily Challenges**: Fresh challenges every day to keep motivation high
- **Streak Multiplier**: Consecutive days of use multiply XP gains
- **Activity Heatmap**: GitHub-style contribution heatmap for productivity tracking
- **Browser Push Notifications**: Native push notifications for timers and reminders
- **Custom Notification Sounds**: Configurable alert sounds via Web Audio API
- **Idle Detection**: Detects user inactivity and pauses tracking
- **Welcome Back Screen**: Personalized greeting after absence with stats summary
- **Confetti Animations**: Celebration effects on achievements and level-ups
- **Landing Page**: Public-facing marketing page with feature showcase
- **Pricing Page**: Tiered pricing display (Free / Pro / Team)
- **Privacy Policy**: GDPR-compliant privacy policy page

### Changed
- Complete UI overhaul with enhanced glassmorphism effects
- Improved animation system with reduced motion support
- Better localStorage data management with versioned schema

### Fixed
- Timer accuracy improvements for long sessions
- Theme transition flickering on initial load

## [1.0.0] - 2025-03-15

### Added
- **Dashboard**: Central hub with widget-based layout
- **Focus Timer**: Pomodoro-style timer with configurable work/break intervals
- **Kanban Tasks**: Drag-ready task management with To Do / In Progress / Done columns
- **Rich Notes**: Markdown-compatible note-taking with auto-save
- **Quick Links**: Customizable link grid for frequently visited sites
- **Ambient Sounds**: Background soundscapes (rain, forest, cafe) via Web Audio API
- **Command Palette**: Ctrl+K quick action launcher
- **Dark/Light Theme**: Toggle with system preference detection
- **Responsive Design**: Full mobile and tablet support
- **Docker Support**: nginx:alpine container with docker-compose
- **Keyboard Shortcuts**: Number keys (1-5) for view switching, Space for timer
